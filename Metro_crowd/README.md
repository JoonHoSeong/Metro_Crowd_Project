## AI기반 지하철 객차별 혼잡도 제공앱


AI모델을 활용하여 사람의 수를 Count 하여 혼잡도를 계산, 웹에서 계산된 혼잡도를 활용하여
실제 사용자가 본인이 타게될 열차의 혼잡도를 볼 수 확인 할 수 있게 만들어주는 웹페이지 구성을 위한 프로젝트 입니다.

## 진행중
